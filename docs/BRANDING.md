# MIBU Branding

MIBU is part of the THETECHGUY TOOL family.

## Public name

- App name: MIBU
- Full name: Mi Bootloader Unlock Helper
- Product family: THETECHGUY TOOL
- Footer line: By the THETECHGUY TOOL team

## Brand breadcrumb rule

Every MIBU screen should quietly point back to the wider tool family.

Recommended places:

- APK launcher name: MIBU
- Android first screen: MIBU with large ghost hero artwork
- Android footer: By the THETECHGUY TOOL team
- PC helper window title: MIBU PC Helper
- PC helper footer: THETECHGUY TOOL
- Logs header: MIBU / THETECHGUY TOOL

## Visual direction

- Use the MIBU logo as the launcher icon.
- Use the large ghost artwork as the main in-app hero.
- Keep the UI dark and tool-like.
- Avoid clutter. This should feel like a professional service utility, not a random script wrapper.

## Copy tone

The tone should be clear, technical, and careful:

- Show what step the user is on.
- Explain what the PC helper is doing.
- Explain what the phone app is waiting for.
- Convert Beijing time to the user's local time automatically.
- Warn that bootloader unlocking wipes data and may affect warranty.
