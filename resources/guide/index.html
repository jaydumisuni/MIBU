Prepare on PC. Continue on the phone.

MIBU checks ADB, installs and opens the Android app, guides the explicit Xiaomi browser session handoff, then transfers the countdown to the phone so the PC is no longer needed while waiting.

Before starting: back up the phone. A bootloader unlock can erase data and may affect warranty or device security. MIBU never asks for your Xiaomi password.

One-Click Assist

1

Connect the phone

Use USB, enable Developer options, OEM unlocking and USB debugging, then accept the RSA prompt with Always allow from this computer.

2

Open the MIBU assistant

Click the floating assistant, then press Start One-Click Assist on its chest. The helper checks the real device state and installs or updates MIBU without opening a terminal window.

3

Approve only phone-side security prompts

Xiaomi may require one-time approval for USB debugging or installing through USB. The assistant pauses and tells you exactly what must be accepted.

4

Complete the browser handoff

The helper opens Firefox and Chrome. Sign in to the same Xiaomi account yourself, collect the two approved session values, and push them to MIBU.

5

Start waiting on Android

MIBU verifies both captures, shows Beijing and local time, arms the foreground service and updates the visible countdown. Keep mobile data available near the target.

Firefox: lanes 1 and 3

Open the Xiaomi Community page while signed in. In Cookie Editor, locate new_bbs_serviceToken and paste its value into the Firefox field in MIBU PC Helper.

Chrome: lanes 2 and 4

Open the same Xiaomi account in Chrome and collect popRunToken. Paste it into the Chrome field. Captures are short-lived, so collect both close to the target window.

Phone handoff and countdown

1

Confirm session ready

Both Firefox and Chrome captures must show ready.

2

Confirm network

Open MIBU Settings, keep a valid SIM inserted, turn off Wi-Fi if required by the Xiaomi flow, and verify mobile data is active.

3

Tap Start Waiting

The service notification and dashboard must show RUNNING. The countdown updates every second and survives leaving the app.

4

Verify the official result

At the end of the timing stage, continue with Xiaomi's official account/device binding and Mi Unlock checks. Record only the result the official tool actually shows.

Manual recovery

Device remains waiting

Reconnect USB, unlock the phone screen, accept the RSA prompt, and run Device Check. Try a data-capable cable or another USB port if no serial appears.

Install pauses

Approve Xiaomi's Install via USB or Shell installer prompt on the phone, then press Install APK again.

Session is partial

Collect both approved values again. Firefox fills lanes 1 and 3; Chrome fills lanes 2 and 4. Do not mix Xiaomi accounts.

Timer does not arm

Open Android Settings inside MIBU, reset an incorrect recorded result if necessary, import fresh captures, allow notifications, and tap Start Waiting again.
